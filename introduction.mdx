SlideMenu is a SwiftUI library that adds a native-feeling slide-out side menu to any iOS app. You wrap your root view in a single `SlideMenu` container, and the library handles edge-swipe gestures, tap-to-dismiss, and visual transitions — no bindings, no delegates, no third-party dependencies required.

## How it works

SlideMenu wraps your main content and your menu content in a single container view. When the user swipes from the screen edge — or taps a `SlideMenuButton` — the main surface slides aside to reveal the menu panel behind it.

The toggle action is injected into the SwiftUI environment automatically. Any `SlideMenuButton` you place inside your main content reads that action from the environment, so you never pass a binding or closure manually. The same `\.slideMenuToggle` environment key is available inside the menu content too, which lets menu items close the drawer after navigating.

All visual properties — width, background material, overlay opacity, border, and animation — are set through view modifiers applied to the `SlideMenu` or any ancestor view.
